# UA-004 活塞滴定管交付滴定剂（读数差法）

## 场景
用 25 mL A 级活塞滴定管滴定，记录初始读数与终点读数，交付体积为两者差值。读数到 0.01 mL。

## 被测量（输出量）
- 输出量：氢氧化钠滴定剂交付体积 $V$（单位：mL）

## 原始数据
- 初始读数：$R_i=0.12 \mathrm{mL}$
- 终点读数：$R_f=18.47 \mathrm{mL}$
- 体积估计值：$\hat V = R_f-R_i = 18.35 \mathrm{mL}$

## 已知误差信息
- 滴定管允差（A 级，25 mL）：$\pm 0.05 \mathrm{mL}$（矩形分布；作为整管校准误差，作用于 $V$）
- 读数分辨力：$d=0.01 \mathrm{mL}$（矩形分布，半宽 $d/2$；初读数和终读数的量化误差独立）
- 终点附近操作者视差/读数判断：每次读数额外误差限 $\pm 0.02 \mathrm{mL}$（矩形分布；两次读数独立）
