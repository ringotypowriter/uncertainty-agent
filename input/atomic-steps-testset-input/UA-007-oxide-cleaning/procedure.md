# UA-007 金属表面氧化层清理（酸洗造成的质量损失）

## 场景
高纯金属片表面酸洗去除氧化层。酸洗会引入少量基体金属溶解损失，需要评定“酸洗导致的质量损失”以用于后续质量校正。

## 被测量（输出量）
- 输出量：镁条表面氧化镁酸洗清理导致的质量损失 $\Delta m$（单位：mg）

## 原始数据
同一块金属片，酸洗前后各称量 3 次（分析天平分辨力 0.01 mg），得到质量（mg）：
- 酸洗前：10234.56, 10234.58, 10234.57
- 酸洗后：10233.91, 10233.90, 10233.92
- 估计损失：$\hat{\Delta m} = \bar m_{before}-\bar m_{after}$

## 已知误差信息
- 天平校准：在 0–20 g 范围，示值误差限 $\pm 0.05 \mathrm{mg}$（矩形分布；对“差值”可视为完全相关并抵消）
- 分辨力：$d=0.01 \mathrm{mg}$（矩形，半宽 $d/2$；对每次称量独立）
