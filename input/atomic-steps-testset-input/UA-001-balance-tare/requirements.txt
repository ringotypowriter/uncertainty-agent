## 评定要求（为保证题目唯一性）
1. 只评定本次记录的 $I_0$ 的标准不确定度 $u(I_0)$。
2. 纳入分量（彼此独立）：
   - Type A：短期噪声为“预评估重复性”。已给出 $n=10$ 次重复读数得到的单次读数实验标准偏差 $s$；由于本题被测量为单次记录的 $I_0$（非 10 次均值），故 Type A 标准不确定度取 $u_A=s$（不再除以 $\sqrt{n}$），自由度取 $\nu=n-1=9$。
   - Type B：分辨力（四舍五入），校准证书给出的 0 g 点误差限，零点漂移
3. 分辨力按四舍五入到最接近末位：量化误差半宽 $d/2$，矩形分布。
4. 给出合成标准不确定度 $u_c$；扩展不确定度不作为必需评分项，若给出可取 $k=2$ 并应满足 $U=2u_c$。
