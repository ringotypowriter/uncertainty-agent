# UA-017 溶剂氮吹浓缩（浓缩至目标终体积）

## 场景
提取液在 40°C 水浴下氮吹浓缩至目标终体积 1.00 mL。终体积通过刻度微量离心管读数控制。

## 被测量（输出量）
- 输出量：乙腈提取液氮吹浓缩后终体积 $V_f$（单位：mL）

## 原始数据
- 目标刻度：1.00 mL
- 实际读数：$R=1.00 \mathrm{mL}$（读到 0.02 mL）

## 已知误差信息
- 微量管刻度允差：$\pm 0.05 \mathrm{mL}$（矩形）
- 读数分辨力：$d=0.02 \mathrm{mL}$（矩形，半宽 $d/2$）
- 终点判断（“近干”提前/滞后）：操作导致的体积偏差限 $\pm 0.03 \mathrm{mL}$（矩形）
