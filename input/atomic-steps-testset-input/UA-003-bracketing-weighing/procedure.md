# UA-003 高精度布氏法称量（Bracketing/比对称量）

## 场景
同位素稀释质谱（IDMS）用高纯金属标准样品需要高精密度称量。使用比较天平进行布氏法称量：用校准砝码 $m_s$ 夹持比较样品 $m_x$。

称量序列（同一环境、同一日内完成）：
1) 砝码（S）
2) 样品（X）
3) 砝码（S）

## 被测量（输出量）
- 输出量：高纯锌标准样品质量 $m_x$（单位：g）

## 原始数据
- 比较天平读数为质量差示值（单位：mg），正值表示“被称物更重”。
- 砝码名义质量：$m_s=10.000000 \mathrm{g}$
- 砝码校准证书：
  - 校准值：$m_s=10.000012 \mathrm{g}$
  - 标准不确定度：$u(m_s)=0.000004 \mathrm{g}$（正态，$k=1$）
- 三次比较差值读数（mg）：
  - $\Delta_1$（S→X）：$+0.213 \mathrm{mg}$
  - $\Delta_2$（X→S）：$-0.205 \mathrm{mg}$
- 比较天平分辨力：$d=0.001 \mathrm{mg}$（矩形分布，半宽 $d/2$）
- 比较过程的重复性：单次差值读数的标准偏差 $s_\Delta=0.004 \mathrm{mg}$（正态）
