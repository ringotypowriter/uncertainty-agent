# UA-002 金属样品的减量法称量（减量法）

## 场景
高纯金属小块在称量瓶中称量。为降低天平示值系统误差影响，采用减量法：先称“容器+样品”，倒出样品后再称“容器”。两次称量在 3 min 内完成。

## 被测量（输出量）
- 输出量：高纯锌样品质量 $m$（单位：g）

## 原始数据
- 天平：分析天平，分辨力 $d=0.0001 \mathrm{g}$
- 读数 1（容器+样品）：$I_1=28.6432 \mathrm{g}$
- 读数 2（仅容器）：$I_2=23.5176 \mathrm{g}$
- 样品质量估计值：$\hat m = I_1-I_2=5.1256 \mathrm{g}$

## 已知误差信息
- 校准证书：在 0–50 g 范围内，示值误差限 $\pm 0.00020 \mathrm{g}$（矩形分布；对每次读数单独适用，且两次读数的校准误差视为完全相关）
- 分辨力：每次读数均存在量化误差，半宽 $d/2$（矩形分布；两次读数的分辨力误差互不相关）
- 称量重复性：在该载荷附近单次称量重复性标准偏差 $s_r=0.00008 \mathrm{g}$（正态分布；两次称量互不相关）
