# UA-020 逐级稀释配制标准系列（由母液配 0.500 mg/L）

## 场景
从 1000 mg/L 母液配制 0.500 mg/L 标准溶液，采用两步稀释：
1) 用 1.00 mL 移液管取母液至 100.00 mL 容量瓶，得中间液 $C_1$。
2) 用 5.00 mL 移液管取中间液至 100.00 mL 容量瓶，得目标液 $C_2$。

## 被测量（输出量）
- 输出量：硝酸盐标准目标溶液浓度 $C_2$（单位：mg/L）

## 输入数据
- 母液证书值：$C_0=1000.0 \mathrm{mg/L}$，$u(C_0)=2.0 \mathrm{mg/L}$（正态）
- 1.00 mL 移液管允差：$\pm 0.006 \mathrm{mL}$（矩形）
- 5.00 mL 移液管允差：$\pm 0.010 \mathrm{mL}$（矩形）
- 100.00 mL 容量瓶允差：$\pm 0.08 \mathrm{mL}$（矩形；两只容量瓶独立）
