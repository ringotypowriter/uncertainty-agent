# UA-018 过滤与多次洗涤（沉淀转移损失分数）

## 场景
对沉淀进行过滤并用热洗液洗涤 3 次。为评定过滤/洗涤导致的目标物损失，采用示踪加标：沉淀中初始目标量为 $M_0$，过滤后滤液+洗液中测得损失量 $M_L$，损失分数 $f=M_L/M_0$。

## 被测量（输出量）
- 输出量：氯化银沉淀过滤与洗涤过程的损失分数 $f$（无量纲）

## 原始数据
- 初始目标量估计：$M_0=100.0 \mathrm{mg}$
- 滤液+洗液中测得目标量：$M_L=2.6 \mathrm{mg}$

## 已知误差信息
- $M_0$ 的标准不确定度：$u(M_0)=0.8 \mathrm{mg}$（正态）
- $M_L$ 的标准不确定度：$u(M_L)=0.2 \mathrm{mg}$（正态）
